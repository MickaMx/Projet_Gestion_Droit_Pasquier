Présentation

Cette application de gestion des droits est destiné à l'entreprise Pasquier. Sur l'application, les utilisateurs peuvent rechercher des autorisation suivant plusieurs critères comme le nom. Sur le tableau des résultats, il peut choisir de modifier une autorisation ou d'en créer.
Sur la page de création d'autorisation, l'usager peut choisir les personnes comprisent dans celle-ci. Il faut ensuite créer le droit liés a l'autorisation en sélectionnant a quoi elle s'applique.


Le but de cette alternance est de s'impliquer dans la vie de l'entreprise. Pour réaliser ce projet, beaucoup de compétences ont été utilisées : programmation objet, algorithmie et langage de programmation. Ce fut un challenge, car beaucoup de travail fut nécessaire pour développer cette application.

Technologies et Outils

Microsoft visual studio

Microsoft Visual Studio est un environnement de développement intégré (IDE) complet pour Android, iOS, Windows, Internet et le cloud. Visual Studio utilise les plateformes de développement Microsoft comme Windows API ou encore Windows Forms.

Framework .NET Core

.NET Core Framework est un cadre applicatif développé par Microsoft. Il permet l'utilisation de nombreuses librairies à travers plusieurs langages de programmation. Le framework est destiné à être utilisé par la plupart des nouvelles applications créées pour la plateforme Windows.

Javascript

JavaScript est un langage de programmation de scripts principalement employé dans les pages web interactives mais aussi pour les serveurs2 avec l'utilisation (par exemple) de Node.js. C'est un langage orienté objet à prototype, c'est-à-dire que les bases du langage et ses principales interfaces sont fournies par des objets qui ne sont pas des instances de classes.

Vue.js

Vue est un framework progressif pour la construction d'interfaces utilisateur. Contrairement à d'autres framework monolithiques, Vue est conçu de A à Z pour être adopté progressivement. La bibliothèque de base se concentre uniquement sur la couche vue et est facile à récupérer et à intégrer avec d'autres bibliothèques ou projets existants.

Travaux

Architecture


En temps qu’application web, le projet peut être découpé en deux parties distinctes :
L'interface utilisateur : Le visuel de l'application, qui offre à l'utilisateur une interface pour agir sur les données. Développé grâce à Vue.js.
La couche d'acces aux données : traite les données pour effectuer les requêtes et implémente les différentes fonctionnalités.



Développement

Cette phase s’est effectuée une fois la conception réalisée.
Ce temps de développement a permis la réalisation d'un logiciel comprenant plusieurs fonctionnalitées. Le premier permet la recherche d'autorisations suivant plusieurs critères, puis différentes actions sur les résutats obtenus. La deuxième fonctionnalitées est la création d'autorisation. celle-ci comprend la création du groupe de personnes puis du droit en question.
